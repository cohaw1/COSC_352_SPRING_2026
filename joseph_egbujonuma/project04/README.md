this project covers single and multi thread programming via a prime number identifier and counter all done in 3 languages

Languages used : 
https://kotlinlang.org/Links to an external site.
https://go.dev/Links to an external site.
https://www.java.com/en/


the test data that was used for this project is named "numbers.txt"

prerequisites: 
- Java JDK (for `javac` + `java`)
- Kotlin compiler (`kotlinc`)
- Go toolchain (`go`)
- Bash


------------How to run-------------
From the project folder, run:
chmod +x run.sh
./run.sh numbers.txt
